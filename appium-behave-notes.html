to setup:

$ pip install -r requirements.txt

to execute:

setup virtual environment

$ virtualenv venv
$ source venv/bin/activate

install python libraries

$ pip install -r requirements.txt

start appium -> connect to android / genymotion appium

$ python behave
$ behave ./features/HKOApp_9DayForecast.feature
